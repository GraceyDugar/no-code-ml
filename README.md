# No-Code ML

AutoML web app where users upload any CSV dataset 
and train ML models through a simple UI — no coding required.

## Features
- Upload any CSV dataset
- Select target column to predict
- Choose between Random Forest and Logistic Regression
- See accuracy, feature importance chart and detailed report

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## How to Run
pip install streamlit pandas scikit-learn matplotlib seaborn
streamlit run app.py
